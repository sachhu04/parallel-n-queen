Parallel N-Queens Demonstration

Visualizing Threaded Execution

Recommended: N between 4 and 10. (N > 10 is slow to generate)

This simulation demonstrates how your C++ OpenMP code parallelizes the N-Queens problem. For N=4, the main loop #pragma omp parallel for divides the work into 4 initial tasks:

  • Task 1: Find all solutions starting with a queen at (0, 0).
  • Task 2: Find all solutions starting with a queen at (0, 1).
  • Task 3: Find all solutions starting with a queen at (0, 2).
  • Task 4: Find all solutions starting with a queen at (0, 3).

Each card below represents a thread (or task) exploring one of these starting positions. Click "Start" to see them work in parallel.
